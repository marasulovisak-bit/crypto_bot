# COMMAND CENTER

## ChatGPT — Технический директор

Отвечает за:

- стратегию проекта;
- распределение задач;
- проверку работы других ИИ;
- архитектуру;
- принятие решений.

## Оператор (Марис)

Отвечает за:

- связь между агентами;
- отправку команд;
- подтверждение решений;
- хранение проекта.

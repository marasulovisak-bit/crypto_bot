# CLOUD WORKSPACE

## Цель

Проект не должен зависеть от одного компьютера.

Работа должна быть доступна с ПК и телефона.

---

## Основные облачные сервисы

### GitHub

Хранение кода и документов.

### Firebase Studio

Облачная среда разработки.

### Android Studio Cloud

Удаленная сборка и тестирование.

### Google Drive

Резервное хранение.

---

## Тестирование

### Android

Эмулятор Android.

### Desktop

Windows.

### Web

Chrome.

---

## AI-команда

Архитектор:
- ChatGPT

Диспетчер:
- Gemini Flash

Разработка:
- DeepSeek

Тестирование:
- Qwen

Дизайн:
- Figma

Хранилище:
- GitHub

---

## Работа оператора

Марис общается только с ChatGPT.

ChatGPT управляет всей системой.

---

## Аварийный режим

При отключении компьютера работа продолжается через телефон.

Проект восстанавливается через GitHub.

История изменений сохраняется.

---

## Главный принцип

Проект должен работать независимо от:

- одного компьютера;
- одного ИИ;
- одного устройства.

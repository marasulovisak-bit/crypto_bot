# Главная цель

Создать облачного AI-ассистента для торговли криптовалютой.

Основные функции:

- Подключение кошельков.
- Стратегия «Разгон депозита».
- Расчёт уровней покупок и продаж.
- Управление риском.
- Статистика.
- Push-уведомления.
- Сравнение с HODL.
- Бэктестинг.
- Работа через облачных ИИ-агентов.
